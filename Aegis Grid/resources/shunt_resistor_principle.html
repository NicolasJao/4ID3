FEEDER LINE — SERIES + SHUNT RESISTOR NETWORK

Click any series resistor (Rs) to simulate a line break at that point

How it works: Series resistors Rs divide the feeder into sections, creating measurement nodes. Shunt resistors Rsh connect each node to ground — current through them creates a voltage drop you can measure. Nodes past a break show zero current → zero voltage drop, so the last "live" node before zero identifies the fault section.
